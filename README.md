# Mastoras.com

Πλατφόρμα εύρεσης επαληθευμένων τεχνιτών στην Ελλάδα.

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS v4
- MongoDB + Mongoose
- Stripe Connect (escrow payments)
- NextAuth.js

## Getting Started
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Project Structure
```
src/
├── app/          # Pages & API routes
├── components/   # UI Components
├── lib/          # DB, Stripe, Auth config
├── models/       # Mongoose models
└── types/        # TypeScript types
